The workbench is a functional block that is used to craft [[:Category:Tools|Tools]].

== Obtaining ==

The workbench can be crafted with 4 of any type of [[:Category:Planks|planks]].

== Usage ==
Interacting with the workbench block (<code>placeBlock</code>) opens the Workbench UI, which can be used to craft the [[Pickaxe]], [[Axe]], [[Shovel]], [[Sickle]], or [[Chisel]]. Clicking the button on the right allows cycling through the different tools.

For more info see [[Procedural Crafting System]]
